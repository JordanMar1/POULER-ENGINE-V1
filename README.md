# Pouler Engine

Experience one of the easiest game engine on the market

## ✨ Features of Pouler Engine

- **Customisation**: Thanks to WAC format you can change easily a Map, a Weapon, an UI Item and even the Menu or Player stats
- **A 3D Raycaster**: Thanks to a Heatmap you can have up to 9 different heights on a map with teleporters
- **Debbuger**: Conceived with an included Debugger
- **Multiple mod files (one per game without assets included)**: you can have multiple mods on a same folder (even if I would not advise it) 
- **C++**: you can easily change a function which does something if you don't like it
- **CSFML**: Port of the SFML on C used for this motor, I will in the V2 change it to SFML
- **Gzdoom open for everyone**: Even if it doesn't supports *id tech 2* it tries to replicate it's versatility and it's small size (5.6 MB actually)


## ✨ How to launch it

```bash
$> make
  ...
$> ./pouler_or_not -f path/to/your/config.wac -d #if you want debbuger activated
```
